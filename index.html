Academic HTML for Node.js and the browser.

Enhance HTML5 with features inspired by TeX—automatic numbering, reciprocal links, easy figure references, pluggable citation styles, and more.

See features

Try it. Edit some HTML:

Ouput

Auto-numbering

Figures and notes are automatically numbered. You can choose digits, roman, or latin styles.

<endotes latin/>

Auto-linking

In-text citations link to full-length citations, end notes link reciprocally, and figure numbers link to figures.

<citation smith89/>

Easy references

Use the simple tag ID feature and you can forget maintaining page anchors from text to figures.

<diag map-usa/>

Citation styles

Use MLA or APA styles out of the box, or create a custom style that suits the conventions in your discipline.

^AUTHOR (YEAR). <i>TITLE</i>. PUBLISHER.

Transpile on the server or client.

It’s best to transpile on the server if you can, but the js file is small (≈5.5KB zipped) and runs fast. Even budget mobiles can be expected to perform pretty well.

In node.js:

npm install labcoat --save

Then:

var http = require('http')
var labcoat = require('labcoat')

http.createServer(function (req, res) {
  res.end(labcoat('<html>…'))
}).listen(3000)

In the browser:

  ⋮
  <script src="/js/labcoat.min.js"></script>
  <script>
    document.addEventListener('DOMContentLoaded', function () {
      document.body.innerHTML = labcoat(document.body.innerHTML)
    })
  </script>
</body>
