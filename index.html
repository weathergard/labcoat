Academic HTML for Node.js and the browser.

Enhance HTML5 with features inspired by TeX—automatic numbering, reciprocal links, easy figure references, pluggable citation styles, and more.

See features

Try it. Edit some HTML:

Input
Ouput

Auto-numbering

Figures and notes are automatically numbered. Choose from digits, roman, or latin numbering styles.

<endotes latin/>

Auto-linking

In-text citations link to full-length citations, end notes link reciprocally, and figure numbers link to figures.

<citation smith89/>

Easy references

Use the simple tag ID feature and stop maintaining page anchor links from body text to element ID attributes.

<diag mygraph/>

MLA & APA Styles

Use MLA or APA styles out of the box, or easily create custom styles that suit the conventions in your discipline.

<bibliography mla>

Transpile on the server or client.

In the browser:

The file is small (5k gzipped) and executes extremely fast. Put this before the closing body tag, and you're done: 

<script src="/js/labcoat.min.js"></script>

In node.js:

npm install labcoat --save

var http = require('http')
var labcoat = require('labcoat')

http.createServer(function (req, res) {
  res.end(labcoat('<html>…'))
}).listen(3000)
